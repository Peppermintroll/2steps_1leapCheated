# Two Steps Forward, One Million Leaps Below

## An unofficial cheat mod for Social Democracy: An Alternate History

Cheat List

- Maintaining dues gives you 1000000 resources
- cutting all taxes gives 1000000 budget
- campaigning towards workers gives 100% vote share for all classes
- Both the internal police and prussian police can be made disturbingly loyal with a single action. [The increase loyalty action, duh]
